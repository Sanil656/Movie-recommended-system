# Movie-recommended-system
This Movie Recommender System is a content-based machine learning project built using Python and Streamlit. It recommends the top 5 similar movies based on user selection using cosine similarity. Movie posters are fetched in real-time using the TMDB API, providing an interactive and user-friendly web application experience.
